## 1. HTML

### 1.1 如何理解 HTML 语义化

- 让人容易读懂，例如 `<article>xx</article>` article 元素一看就知道当前内容代表是文章。
- 让搜索引擎更容易读懂，还是上面的例子，有利于搜索引擎爬虫爬取有效信息，提高站点的排名。
- 在没有 css 的情况下，页面也能呈现出很好地内容结构、代码结构
- 利于程序的开发与维护

### 1.2 Script 标签 async 和 defer 区别

| script 标签        | js 执行顺序                        | 是否阻塞解析 HTML                       |
| ------------------ | ---------------------------------- | --------------------------------------- |
| `<script/>`        | 顺序执行                           | 阻塞                                    |
| `<script async />` | 网络请求返回后立即执行(按返回顺序) | 可能阻塞，可能不阻塞(已经解析完了 HTML) |
| `<script defer />` | 顺序执行                           | 不阻塞                                  |

### 1.3 浏览器地址从输入 url 到按下回车发生了什么

DNS 解析过程 + 响应过程

DNS 解析([参考文章](https://juejin.cn/post/6990344840181940261))：

- 检查 DNS 缓存，过程：浏览器 => 系统 => 路由器 => ISP(本地网络服务商，如接入电信网络则电信部门就是 ISP)，如果缓存存在，直接返回 ip 地址。
- 如果缓存不存在，则向公共域名解析服务器发起 DNS 查找请求，过程：根 DNS 服务器 => 顶级域 DNS 服务器(TLD) => 权威 DNS 服务器，最终返回 域名-ip 映射
