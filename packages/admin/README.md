#### 安装教程

1.  `git clone https://gitee.com/flying_purchase/dashboard/projects/flying_purchase/YangTao-Admin`
2.  `cd YangTao-Admin`
3.  `npm i`

1.  `npm start`：启用开发模式
2.  `npm run build`：打包
